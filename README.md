## Projects

**[ERCOT Load Analysis](https://github.com/SEON97UN/ercot-load-analysis)**  
Texas grid demand analysis (2019–2024) via EIA API. Context-aware anomaly detection
by season × hour × day type, compared against Isolation Forest.
Includes Winter Storm Uri case study. Python · pandas · scikit-learn

**[InvestDNA](https://investdna.pages.dev)** — *live*  
Bilingual (KR/EN) web app that maps investment behavior to one of 8 archetypes across a 3-axis personality model. React + Vite + Tailwind, deployed on Cloudflare Pages.

**[Time To Buy Pro](https://github.com/SEON97UN/time-to-buy-pro)** — *in progress*  
Quantitative investing dashboard for retail investors. Market regime detection, historical pattern matching, walk-forward backtesting, and portfolio P&L tracking in one workflow. Python + FastAPI backend, Vite + React + TypeScript frontend, Supabase, Telegram alerts.

**[Multi-Strategy Stock Trading Analysis](https://github.com/SEON97UN/multi-strategy_stock_trading_analysis)**  
Quantitative research platform comparing five systematic strategies — PCA-based factor model, momentum/mean reversion, statistical arbitrage, Random Forest/XGBoost classification, and LSTM with attention — under a shared backtesting framework. Interactive Streamlit dashboard for strategy testing and result visualization.

**[MBTI-based Media Recommendation](https://github.com/MVTI-MovieAndVideo-Recommender-Platform)**  
Hybrid recommendation system that embeds Korean VOD content and MBTI personality posts into the same vector space using LaBSE, then combines cosine similarity matching with K-means clustering and GBM predictions. Evaluated at NDCG@20: 0.6206.
